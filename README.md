# Jolt-API

## Description

Jolt-API est une API RESTful pour la gestion des utilisateurs, permettant de récupérer, rechercher et gérer les informations des utilisateurs.

## Prérequis

- Node.js
- MongoDB
- npm

## Installation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/votre-utilisateur/jolt-api.git
   cd jolt-api
   ```
